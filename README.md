# QuickSlides - Instant Elegant Presentations

Create beautiful presentations in seconds without wasting time on formatting!

## Features

### Smart Content Detection
- Automatically detects titles, subtitles, and bullet points
- No special formatting required - just paste your text
- Intelligently organizes content into slides

### Content-Related Images
- Automatically fetches images related to your slide content
- Uses Unsplash API to find relevant, high-quality images
- Toggle on/off with the "Auto-add related images" checkbox
- Or paste your own image URLs directly in the content

### Multiple Themes
- **Modern**: Purple gradient with elegant styling
- **Minimal**: Clean white background with subtle accents
- **Dark**: Dark mode with cyan highlights
- **Gradient**: Pink gradient for vibrant presentations
- **Custom**: Upload your own CSS template!

### Export Options
- **PDF**: Print to PDF for sharing
- **PowerPoint (PPTX)**: Real PowerPoint files that work in MS Office
- **HTML**: Standalone HTML file with navigation
- **Text**: Plain text format for editing

### Custom Templates
Upload templates in multiple formats:

**CSS Template** (`.css`)
1. Create a CSS file with your custom styles (see `sample-template.css`)
2. Click "üìÅ Upload Template" button
3. Select your CSS file
4. Choose "Custom Template" from the theme dropdown

**JSON Template** (`.json`)
1. Create a JSON file with theme settings (see `sample-template.json`)
2. Upload it using the template button
3. Easier than CSS - just define colors and fonts!

**Background Image** (`.jpg`, `.png`)
1. Upload any image file
2. It becomes your slide background
3. Text automatically gets shadows for readability

Generate slides to see your custom design!

## How to Use

1. **Open** `index.html` in your web browser
2. **Paste** your content in the text area
3. **Choose** a theme from the dropdown
4. **Click** "Generate Slides"
5. **Navigate** with arrow keys or buttons
6. **Export** in your preferred format

## Content Format Tips

### Automatic Detection
Just paste plain text and QuickSlides will organize it:
- Short, capitalized lines become titles
- Medium lines after titles become subtitles
- Lines starting with -, *, ‚Ä¢, or numbers become bullets

### Manual Formatting
For more control, use markdown-style formatting:
```
# Main Title
## Subtitle

- Bullet point 1
- Bullet point 2

---

# Next Slide Title
Content here...
```

### Adding Images
- Paste image URLs directly in your content
- Or enable "Auto-add related images" for automatic image selection
- Images are automatically styled and positioned
